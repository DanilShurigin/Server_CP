Подробное описание

Класс для управления сетевым взаимодействием.

Создаёт сокет и закрепляет его. Запускает режим прослушивания для приёма запросов от клиентов. Принимает соединение с клиентами.

Конструктор для создания и запуска сетевой части сервера.

Аргументы
[in]portПорт, на котором будет работать сервер.
[in]queue_lengthКоличество клиентов, которые могут одновременно ожидать подключения.
[in]viewerСсылка на объект ClientBase.
[in]loggerСсылка на объект LogWriter.
[in]max_threadsМаксимальное число потоков, в которых будут обрабатываться клиенты.
Исключения
ServerExceptionОшибки при запуске сетевых функций. Параметры исключений:
what = "Failed to create socket."
"Failed to set socket options (SO_REUSEADDR)."
"Failed to bind socket."
"Failed to start listen."

◆ operator()()

void Communicator::operator() ( )

Приём запросов от клиентов.

В однопоточном режиме принимает запрос следующего в очереди клиента после обработки предыдущего. В многопоточном режиме принимает новые запросы, если свободен хотя бы один поток.
